**Hotel Manager** — это современное веб-приложение для автоматизации работы отеля, управления номерным фондом, бронированиями и персоналом. Система построена на высокопроизводительном стеке FastAPI + HTMX, что обеспечивает "одностраничный" пользовательский опыт без перегрузки страниц.

---

## ✨ Ключевые возможности

### 🏨 Управление номерами
- **Интерактивный дашборд**: визуальное отображение состояния всех номеров.
- **Статусы комнат**: Убрана, Требует уборки, На ремонте, Занята.
- **Фильтрация**: быстрый поиск по статусу (свободные, грязные, на брони и т.д.).
- **Детальные карточки**: полная информация о госте, сроках проживания и ответственном персонале.

### 📅 Система бронирования
- **Календарь занятости**: наглядная сетка бронирований на месяц.
- **Разделение понятий**: четкое различие между "Бронью" и фактическим "Заселением".
- **Интеграция с Flatpickr**: удобный выбор дат с поддержкой диапазонов.

### 👥 Управление персоналом
- **Ролевая модель**: Admin, Moderator, Staff, Cleaner, Worker.
- **Назначение задач**: автоматический выбор типа сотрудника в зависимости от статуса номера (уборка/ремонт).
- **Личные профили**: управление контактными данными и правами доступа.

### 🤖 Telegram Уведомления
- **Моментальные оповещения**: Персонал получает сообщения о назначении на уборку или ремонт.
- **Интерактивность**: В сообщении есть кнопка "Работа выполнена" для быстрой смены статуса номера.
- **Гибкая адресация**: Отправка по ID, телефону или username (требуется регистрация).

### 🎨 Дизайн и темы
- **Адаптивность**: полноценная работа на мобильных устройствах.
- **Темы оформления**: поддержка нескольких цветовых схем (Midnight, Luxe Light, Onyx Dark и др.).
- **HTMX Dynamic UI**: обновление контента "на лету" без перезагрузки страницы.

## 🚀 Быстрый старт

### Требования
- Python 3.10 или выше
- PostgreSQL (опционально, по умолчанию используется SQLite)

### Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone <repository-url>
   cd hotel-manager
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   # Windows:
   .\venv\Scripts\activate
   # Linux/macOS:
   source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите приложение:**
   ```bash
   uvicorn app.main:app --reload
   ```

5. **Откройте браузер:**
   Перейдите по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000)
   *Логин по умолчанию:* `admin` / `admin`

## 🛡 Безопасность и роли
Система использует JWT-токены для авторизации. Доступ к функциям ограничен ролями:
- **Admin**: полный доступ ко всем функциям и управлению персоналом.
- **Moderator**: управление номерами и бронированиями.
- **Staff/Cleaner/Worker**: просмотр назначенных задач и обновление статусов.
